# How to Run the Streamlit Dashboard

1. Open a terminal and navigate to the app directory:
   ```
   cd ad_sales_project/app
   ```
2. Run the Streamlit app:
   ```
   streamlit run dashboard.py
   ```
3. The dashboard will open in your browser. Use the sliders to set ad budgets and click 'Predict Sales'.

- The dashboard shows predicted sales, budget distribution, model insights, and a channel recommendation for best ROI.
